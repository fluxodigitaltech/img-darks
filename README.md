# img-darks

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

Banco de imagens da marca **Darks** — suplementos, vestuário e identidade visual. Repositório mantido pela [**Fluxo Digital Tech**](https://github.com/fluxodigitaltech).

## Conteúdo

- Logo e identidade visual
- Imagens de produtos (suplementos)
- Catálogo de roupas

## Como usar

As imagens podem ser referenciadas diretamente via URL raw do GitHub:

```
https://raw.githubusercontent.com/fluxodigitaltech/img-darks/main/<arquivo>
```
